== Description ==
Create a competition and display it within 20sec. Allows you to add a picture and a little description for each competition.

update 1.2:

Bug fix, option to change the text displayed and allow you to delete a picture associated to a competition.

Features

1. Create a competition
1. Widget ready
1. Can be pulled from any blog post as well as the widget
1. Allow admin to upload picture and description
1. email address and Cookie filtering
1. Mailing list available in admin section
1. general options

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the unzip folder into your plugin folder (/wp-content/plugins/)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a competition from the admin page in manage/competition
1. Use the widget to display the competition or use [competition_id]
